# 💳 Credit Card Fraud Detection

## 📌 Problem Statement
Develop a machine learning model to detect fraudulent credit card transactions.

## 🛠️ Approach
- Preprocessed transaction data (scaling, handling imbalance).
- Algorithms used:
  - Logistic Regression
  - Decision Trees
  - Random Forests
- Evaluated using precision, recall, F1-score, and ROC-AUC.

## 📊 Sample Output
### Example Input:
Transaction: Amount = $2500, Location = Overseas, Time = 2 AM

### Predicted Result:
**Fraudulent Transaction**

### Model Performance:
- Random Forest ROC-AUC: **0.96**
- Logistic Regression ROC-AUC: **0.89**
- Decision Tree ROC-AUC: **0.91**
